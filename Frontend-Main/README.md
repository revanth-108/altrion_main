# Altrion Frontend

**New to the project?** Start with the monorepo onboarding guide: [Developer guide](../docs/DEVELOPER_GUIDE.md).

## Run locally

```bash
cd altrion-frontend
npm install
npm run dev
```

Frontend: `http://localhost:5173`

Required local file: `.env` (copy from `.env.example`).

```env
VITE_API_URL=http://localhost:8000/api
```
